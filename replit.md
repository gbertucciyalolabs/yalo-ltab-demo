# Yalo x LTAB PepsiCo Demo

## Overview
Interactive demo application showcasing WhatsApp commerce journeys for PepsiCo products using Yalo's platform. Built with React + Vite.

## Tech Stack
- React 18
- Vite 5
- Framer Motion

## Structure
- `src/App.jsx` - Main app with journey state management
- `src/components/` - UI components (PhoneMockup, WhatsAppChat, SahulatApp, ControlPanel, etc.)
- `src/journeys/` - Journey definitions (abandoned cart, app handoff, photo list, smart notifications, voice ordering)
- `src/data/` - Product data and Urdu strings
- `src/styles/` - CSS files (animations, sahulat, whatsapp)

## Running
- Dev server: `npm run dev` on port 5000
- Build: `npm run build`
